=== axes (float32) ===

[0] 左スティック X軸
    左: -1 / 右: +1

[1] 左スティック Y軸
    上: -1 / 下: +1

[2] LT（左トリガー）
    押すと -1 → +1

[3] 右スティック X軸
    左: -1 / 右: +1

[4] 右スティック Y軸
    上: -1 / 下: +1

[5] RT（右トリガー）
    押すと -1 → +1

[6] 十字キー 左 / 右
    左: -1 / 右: +1

[7] 十字キー 上 / 下
    上: -1 / 下: +1


=== buttons (int32) ===

[0]  A ボタン
[1]  B ボタン
[2]  X ボタン
[3]  Y ボタン
[4]  LB（左バンパー）
[5]  RB（右バンパー）
[6]  BACK
[7]  START
[8]  Logicool ボタン
[9]  左スティック押し込み
[10] 右スティック押し込み
